Neural Networks to classify answer boxes

Unlike most of the existing techniques that rely on image processing algorithms to recognize an extracted answer box, we have trained a deep neural network model to classify the answer boxes among confirmed, crossed out, or empty boxes, thus providing room for crossing out a marked answer box, and also removing the rigidness of template preparation for a MCQ test to some extent.
We have also added the rest api for our service which can be found from the api option in the navbar.
